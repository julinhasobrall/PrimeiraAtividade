requirements.txt

O arquivo requirements.txt serve para listar todas as bibliotecas (dependências) que o seu projeto Python precisa para funcionar.

Ele é como uma receita de bolo do seu ambiente virtual — mostra tudo o que precisa ser instalado para reproduzir o mesmo ambiente em outro computador.

Como criar um requirements.txt

No terminal:
pip freeze > requirements.txt

requirements.txt

Normalmente ele fica na raiz do projeto, junto com o manage.py
